Skeduler

Skeduler is an ultimate platform for Home Service Companies. It was an outsourcing project for Incwell Venture. This software is used for Ultimate Job Management and Scheduling. It is based on a micro-service architecture that uses programming languages like Python, Laravel (PHP), ReactJS, and NodeJS. I got a chance to work for 3 months on this project and was able to develop and maintain the micro-components of the system.
