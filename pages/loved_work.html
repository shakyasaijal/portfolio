Loved Project

People have loved some of my projects.The most two loved project is "Commerce FM" and "Flask & React Boilerplate". People have engaged in my codes in GITHUB by forking, giving starts, and creating an issue.


"Commerce FM" is an application which can be used to create single or multi-vendor e-commerce application. Backend was done using Python, Django, Django Rest Framework, and Flask. Similarly, frontend was done using ReactJS, Redux, and Webpack. For building, running, and managing all the networks and applications, this project has used Docker.


On the other hand, "Flask & React boilerplate" is a small application with the setup of Flask as a backend and ReactJS as a frontend library. We can clone the project and start coding directly as the structure has already been developed.
